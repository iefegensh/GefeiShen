  2. OPDR: Order-Preserving Dimension Reduction for Semantic Embedding of Multimodal Scientific Data

    01/2024 — 08/2024

    • Using bash to automatically generate scientific material files and simulation images via the Plane-Wave Reciprocal-Space Interpolated Scattering Matrix (PRISM) algorithm
    • Using Transformer-based models (such as BERT, ViT, and CLIP) via the Hugging Face platform to embed multimodal scientific data (such as images, text, and numbers), obtaining high-dimensional embedding vectors.

    • Utilizing dimensionality reduction methods (such as PCA and MDS) to reduce the dimensionality of the high-dimensional data and comparing the information loss across different dimensions to find the optimal reduction scheme.

    • Constructing a closed-form function to quantify the relationship among spatial dimensionality.
    • Validating this new closed-form function across different models (BERT, ViT, and CLIP), distance metrics (Euclidean, Mahalanobis, and cosine), with two different dimensionality reduction methods (MDS, PCA).
